# Daily News Collector

每日定时收集重要资讯，整理归档到知识库。

## 信息分类

1. **美国时政** - 美国政府变动、政策新规、党派动态、选举相关
2. **美国社会** - 社会热点事件、民生问题、舆论焦点
3. **国际大事** - 战争冲突、地缘政治、经济金融、重要国际事件
4. **科技资讯** - 科技公司动态、产品发布、技术进展、行业报告
5. **热点话题** - 名人发言、病毒式传播内容、热门讨论

## 工作流程

1. 使用 web_search 分别搜索各分类最新资讯
2. 汇总信息，去重，筛选重要内容
3. 生成 README 格式报告

## README 格式要求

```markdown
# 每日资讯 - YYYY-MM-DD

## 🇺🇸 美国时政

- 内容

## 🇺🇸 美国社会

- 内容

## 🌍 国际大事

- 内容

## 💻 科技资讯

- 内容

## 🔥 热点话题

- 内容

---
Generated by Jarvis
```

## 输出路径

`~/Documents/KnowLedgeDatabase/daily-news/{YYYY-MM-DD}.md`

## 注意

- 搜索时效：当天或最近 24 小时内容
- 内容要精炼，每条不超过 2 行
- 筛选真正有影响力的内容
