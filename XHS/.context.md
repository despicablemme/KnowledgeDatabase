# 小红书内容总结工作流

## 触发条件
用户通过 QQ 频道发送小红书帖子链接

## 工作流程
1. 使用 mcporter 获取帖子详情（文字 + 图片 URL）
2. 下载并分析图片获取更多信息
3. 按标准格式总结成 md 文档
4. 保存到 ~/JarvisWorkspace/KnowLedgeDatabase/XHS/
5. 文件名：YYYY-MM-DD_标题摘要.md

## 文档格式
参考：~/JarvisWorkspace/KnowLedgeDatabase/XHS/2026-03-14_伊犁公共交通攻略.md

## 工具
- mcporter: 获取小红书帖子数据
- image: 分析图片内容
- read/write/edit: 文件操作
