## 技术笔记

### 迅雷下载自动化流程
1. `thunder://` URL scheme 打开迅雷并添加磁力
2. osascript + System Events 读取 Thunder UI 元素
3. 获取按钮位置：`osascript -e 'tell app "System Events" to tell process "Thunder" to tell button "立即下载" of window 1 to get {position, size}'`
4.cliclick 点击坐标 + osascript key code 36 (Enter) 触发下载
5. 下载目录默认 ~/Downloads
