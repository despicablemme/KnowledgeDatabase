## 1. Events API token 能做什么

| 能力 | 能不能 | 怎么用 |
|---|---|---|
| **被动接收事件** | ✅ | long-poll `eventsapi.chaturbate.com/events/:user/:token/`，推 `broadcastStart`/`broadcastStop`/`tip`/`chatMessage`/`follow` 等 |
| **查自己账号统计** | ✅ | `statsapi/?username=X&token=*** / token 是否有效 |
| **主动查"某主播在不在播"** | ❌ | Events API **没有** query 接口，只能等事件 |
| **列"我所有关注 + 谁在线"** | ❌ | 也没这个 API |

**Events API 本质是 webhook 不是 query** —— 只能"开播时通知你"，不能"现在查一下"。
