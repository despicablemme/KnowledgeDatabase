# 技术帖子汇总工作流

## 触发条件
用户通过 QQ 频道发送技术帖子链接（软件开发、编程相关）

## 工作流程
1. 获取帖子内容（网页抓取）
2. 分析帖子主题/标签
3. 判断 techDocs 目录下是否已有同主题文档
4. 如果有：合并到已有文档（重复内容提炼，不同内容补充）
5. 如果没有：创建新文档
6. 返回总结存放位置

## 命名规则
- 目录：技术领域（如 Android、iOS、Web、Backend、AI 等）
- 文件：具体主题（如 Activity、ViewModel、NextJS 等）

## 文档格式
参考一般技术文档结构：
- 概述
- 核心知识点
- 代码示例
- 踩坑记录
- 参考资料
